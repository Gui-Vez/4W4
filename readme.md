# Épreuve finale - 4W4 - Conception d'interface et développement Web
### Auteur : Guillaume Vézina

```
Ce thème est sujet d'une épreuve finale en Web du cours 4W4 où l'objectif est d'inclure la liste des cours du TIM à l'aide de Wordpress. Voici la structure utilisée:
-Pour le style css nous avons utilisé Sass
-Les fonctions du thème se trouvent dans «functions.php»
-Les modèle principal est : «index.php»

Le fichier template-atelier a été ajouté afin de créer des éléments de texte dynamiques sur Wordpress via l'extension ACF.
Quelques fichiers Sass ont été modifés pour ajouter de l'animation et de l'interactivité entre éléments du site.

Lien pour accéder à la page github.io: https://gui-vez.github.io/4W4/
